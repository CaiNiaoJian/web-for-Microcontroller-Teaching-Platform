# MCU Learn - 单片机学习平台现代化改造

## 项目概述

MCU Learn 是一个专注于单片机学习的综合性教育平台，致力于为学习者提供优质的学习资源和交流平台。在这次为期一周的现代化升级中，我们对整个平台进行了全方位的改造，从用户界面到功能体验都进行了显著提升。

## 项目详情

### 项目简介
这是一个现代化的单片机学习平台网站，为学习者提供优质的单片机教育资源。网站采用响应式设计，支持多端访问，并集成了丰富的交互动画和视差滚动效果。

### 项目结构
```
├── indexmodern.html     # 现代化主页
├── websiteNow.html      # 网站建设进度页
├── z_css/              # CSS 文件目录
│   ├── modern-style.css    # 主样式文件
│   ├── modern-sources.css  # 资源页样式
│   ├── modern-knowledge.css # 知识分享页样式
│   └── blog-style.css      # 博客页样式
├── z_js/               # JavaScript 文件目录
│   ├── modern-script.js    # 主要脚本文件
│   ├── modern-sources.js   # 资源页脚本
│   ├── knowledge.js        # 知识分享页脚本
│   ├── weather.js         # 天气功能脚本
│   └── emoji-fountain.js   # 表情喷泉特效脚本
├── images/             # 图片资源目录
├── StudySources/       # 学习资源目录
├── pages/             # 页面目录
└── News/              # 新闻资讯目录
```

1. **导航系统**
   - 响应式导航栏设计

2. **首页功能**

4. **交互设计**
   - 滚动触发动画

### 使用说明

1. **环境要求**
   - 现代浏览器（支持 ES6+）
   - 稳定的网络连接
   - 建议分辨率 1920×1080 或以上

2. **本地开发**
   ```bash
   # 克隆项目
   git clone [项目地址]
   
   # 进入项目目录
   cd mcu-learn
   
   # 使用本地服务器运行
   # 可以使用 Live Server 或其他本地服务器
   ```

3. **开发建议**
   - 遵循现有的变量命名系统
   - 保持代码风格统一
   - 确保响应式布局适配
   - 注意性能优化

4. **注意事项**
   - 部分动画效果可能在低性能设备上有性能影响
   - 建议使用现代浏览器访问
   - 确保启用 JavaScript
